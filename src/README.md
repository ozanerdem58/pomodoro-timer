# Pomodoro Timer App

A Pomodoro Timer web application built using **Bolt.new** as a vibe coding tool.

## Features
- 25-minute work timer
- 5-minute break timer
- Start, pause, and reset buttons
- Remaining time displayed in mm:ss format
- Work / Break mode indicator

## Technologies Used
- HTML
- CSS
- JavaScript
- Bolt.new (StackBlitz AI)

## How to Run
Run the project directly in Bolt.new or open `index.html` in a web browser.
